## Why LeetCode 75

It covers exactly what gets asked in interviews without 
being overwhelming. I picked this over random grinding 
because every topic here has a purpose — sliding window 
teaches you to avoid nested loops, two pointers teaches 
you to think about space, DP teaches you to break problems 
into smaller ones.

---

## How I approach each problem

I try to solve it on my own first, even if it takes longer.
Then I look at what I got wrong and write a comment in the 
code explaining why the correct approach works. That way 
the repo is useful to me later, not just a collection of 
copy-pasted solutions.

---

## What I'm noticing so far

Arrays and strings felt manageable. Sliding window is 
where things started getting interesting — knowing when 
to expand and when to shrink the window is the whole game.
Still working through it.
